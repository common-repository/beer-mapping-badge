== Description ==

The Beer Mapping Badge Wordpress Widget

This widget allows a wordpress user to add a Badge that ties their Beer Mapping profile to their wordpress blog.

This is really only for people who are into beer or who are interested in spreading the word about good beer and good breweries.

http://beermapping.com/2007/11/25/wordpress-and-blogger-beer-mapping-widget/


Widget Options:
1. Widget Title (optional): add one or don't.  This text will appear above the widget in the sidebar.
1. Beer Mapping User Name: Add your username.  If you leave this blank you will see a random user everytime you load your sidebar.
1. Show User Information: this is the statistics for your account on the Beer Mapping Project.  Including avatar, links to forum posts and other things.
1. Show your top ten scores: If checked, this will call for a list with links to the reviews of the top ten locations you have reviewed on the Beer Mapping Project.

== Installation ==
1. Unzip the file and upload bmpbadge.php into your wp-content/plugins/ folder.
1. Activate the plugin via the Plugins page in your administration panel.
1. Update and set up your widget via the Presentation > Widgets page.
